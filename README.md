## 5.서비스 소개

### 5.1 서비스 개요
자립준비청년은 아동복지시설이나 위탁가정 보호가 만 18세에 종료되며 홀로서기를 시작하는 청년으로, 주거·일자리·재정 정보 부족 등으로 초기 자립 환경이 매우 취약하다. 보건복지부 조사에서 이들 중 46.5%가 ‘자살을 생각한 적이 있다’고 응답할 만큼 심리적 부담도 크다.
푸름(Pureum)은 이들을 위해 자산·소득·선호 기반의 주거–일자리 조합을 추천하여 초기 자립 과정에서의 정보 탐색 부담을 줄이고 안정적인 출발을 돕는다.

### 5.2 타서비스와의 차별점
- 주거와 일자리 각각을 개별적으로 추천하는 기존 플랫폼과 달리 두 가지를 동시에 고려하여 추천한다.
- LM Studio 기반 로컬 LLM을 사용하여 개인정보 외부 유출 없이 개인 맞춤 설명을 생성한다.
- 자립준비청년의 실제 지원사업을 반영한다. 

### 5.3 구현 내용 및 결과물

#### 1) 주거-일자리 추천 시스템
- 사용자의 선호도와 조건을 분석한 후, 이를 로컬 LLM이 해석 및 평가하여 합리적인 조합을 선별한다.
- LLM은 비용 적합성, 통근 가능성, 직무 적합성을 종합적으로 검토하여 조합을 생성하고 해당 조합이 적합한 근거를 설명으로 제공한다. 
- 모든 AI 연산은 로컬 LLM으로 처리되어 개인정보가 외부로 유출되지 않는다. 

#### 2) 월 소득 및 지출 기반 재정관리 시스템
- 사용자가 선택한 주거-일자리 조합을 바탕으로 월 소득 및 지출을 계산하여 맞춤형 저축 전략을 안내한다. 
- 자립 초기 단계에서 필요한 기초 재정 가이드를 제공할 수 있다. 

### 5.4 구현 방식
**Design** : Figma  
**FrontEnd** :Swift  
**BackEnd** : Spring Boot 3, Java, LM stdio


## 6. 향후 개선 혹은 발전 방안
향후 푸름(Pureum)은 자립준비청년의 실제 생활 전반을 지원할 수 있도록 기능을 확장할 계획이다. 

### 1) 지출 관리 기능 고도화
사용자의 실제 계좌를 연동하여 소비 패턴을 자동으로 분석할 수 있는 기능을 도입할 계획이다. 이를 기반으로 월 단위 지출 한도를 설정하거나 자립정착금을 체계적으로 관리하는 등, 보다 정교한 재정 관리 서비스를 제공할 수 있도록 확장할 예정이다. 

### 2) 실시간 채팅 상담 기능 추가
자립 과정에서 발생하는 정서적, 사회적 문제를 지원하기 위해 채팅 상담 기능을 도입할 예정이다. 사용자는 어플을 통해 심리 상담 기관이나 자립 지원 센터와 바로 연결될 수 있다. 기본적인 문의는 AI 상담이 즉각적으로 응답하고, 필요한 경우 전문 기관으로 연계되는 구조를 구축할 계획이다. 

### 3) 자립준비청년 전세임대, 공공임대 시뮬레이션 기능 추가
주거지원 제도 활용을 돕기 위해, LH 청년전세임대 조건을 바탕으로 지원 가능 금액을 자동으로 산출하고, 보증금 및 이자 납부액을 시나리오별로 비교해볼 수 있는 시뮬레이션 기능을 제공할 예정이다.
